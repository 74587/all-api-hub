赞助商服务教程

本页汇总赞助商与推荐服务在 All API Hub 中的配置教程。选择对应服务后，可以按步骤完成账号添加、API 密钥保存、模型价格查看和凭据导出。

这些教程适用于已经注册或准备注册对应服务的用户。如果还没有安装 All API Hub，请先阅读 快速上手。

选择服务

服务适用场景教程
RunAPI添加 RunAPI 账号，查看模型价格，保存并导出 API 凭据。查看教程
PackyCode添加 PackyCode 账号，查看模型价格，保存并导出 API 凭据。查看教程
星辰AI添加星辰AI账号，查看模型价格，保存并导出 API 凭据。查看教程
