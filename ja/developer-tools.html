開発者と高度なツール

上級ユーザーや開発者向けの、デバッグ、視覚的なカスタマイズ、または詳細なトラブルシューティングのための組み込み補助ツールです。

メッシュグラデーションデバッグツール

メッシュグラデーションデバッグツール は、シェアスナップショット 機能で使用される動的な背景効果をプレビューし、カスタマイズするための組み込みの視覚的デバッグツールです。

主な機能

  • リアルタイムプレビュー: 異なるシード値、パレット、レイアウトの組み合わせによる背景生成結果を即座に確認できます。
  • パレットブラウザ: 内蔵されているすべてのカラーパレットとその16進数コードを確認できます。
  • レイアウトの切り替え: サポートされているすべてのメッシュ変形レイアウトをブラウズできます。
  • オーバーレイシミュレーション: データオーバーレイの表示/非表示を切り替えて、実際の生成結果をシミュレートできます。

アクセス方法

このツールは主に開発とテストを目的としているため、通常のメニューには含まれていません。以下の手順でアクセスできます：

  1. 拡張機能の 「設定」 ページを開きます。
  2. ブラウザのアドレスバーで、options.html の末尾に #mesh-gradient-lab を追加します。
    • 例：chrome-extension://<id>/options.html#mesh-gradient-lab
  3. ページが自動的にデバッグビューに切り替わります。

デバッグとログ

使用中に原因不明の問題が発生した場合は、ブラウザに組み込まれている開発者ツールを使用してトラブルシューティングを行うことができます。

1. バックグラウンドの Service Worker ログ

  • ブラウザの拡張機能管理ページ（chrome://extensions）に移動します。
  • 「デベロッパーモード」 を有効にします。
  • All API Hub のカード内にある 「ビューの検査: Service Worker」 をクリックします。
  • ここでは、バックグラウンドでのリクエスト、自動リフレッシュ、WebDAV 同期の低レベルなログを確認できます。

2. オプションページ/ポップアップのログ

  • 拡張機能のポップアップまたは設定ページ上で 右クリック -> 検証（検査） を選択します。
  • 「Console」 タブに切り替えて、UIレイヤーのログ出力を確認します。
